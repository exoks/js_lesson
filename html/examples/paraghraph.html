=== <p>: tag ===

This the block-level element paraghraph where      extra spaces/tabs are deleted & blank lines are removed, and lines are joined until a new line is needed, But you can use HTML entities to add extra spaces.
NOTE: <p>: tag takes the full webpage width of the page.

=== <pre>: tag === 

        This is the preformatted text that is 
                  displayed as it is, No spaces/tabs are deleted, everything is preserved.
        NOTE: use me for code snippests or where
      spaces does really    =>    matters.
